atomic network = the smallest possible network that is stable and can grow on its own. 

Don't try to launch to everyone. The initial goal is to create the smallest possible network that is stable, engaged, and can provide value to its members, thus overcoming anti-network effects and beginning to grow on its own.

> "When you start a new product with network effects, the first step is to build a single, tiny network that’s self-sustaining on its own." 

> "The target should be on building a tiny, atomic network—the smallest that could possibly make sense—and focus on building density, ignoring the objection of “market size.”
