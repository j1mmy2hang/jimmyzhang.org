Deterministic Code versus Flexible LLM 这两个之间的平衡点是在哪里？

1. 未来会不会 LLM 如此便宜、不要钱，以至于直接用 LLM 就可以？
2. Deterministic code 还有没有其价值，还是说会成为艺术品之类的东西？

然后在想为什么 Deterministic code 不用花钱。本质原因是因为我自己买了芯片，然后代码能在我自己的电脑上跑。LLM 之所以花钱，是因为目前本地 AI 算力还不够。

但是就是想，如果未来本地这些小任务 AI 都能跑了，Code 和 LLM 在最终效果上可能就没什么区别了，还有没有必要追求 Deterministic code？
