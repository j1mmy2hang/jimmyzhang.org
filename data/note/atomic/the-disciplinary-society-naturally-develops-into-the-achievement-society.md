## 系统之外的否定性 --> 系统内部的肯定性

| 规训社会   | 功绩社会      |
| ------ | --------- |
| 系统之外   | 系统内部      |
| 他者     | 内在        |
| 否定性    | 肯定性       |
| 不允许/应当 | 能够        |
| 疯子和罪犯  | 抑郁症患者和厌世者 |

> 不是由免疫学上他者的否定性导致，而是由以一种过量的肯定性引发。
> 新的暴力形式不来自一个系统之外的否定性他者，而源于系统内部。

**正是基于它的内在性，免疫反应对它失去效力。**

E.g. 霍尔顿什么都讨厌，什么都想反抗，但他的反抗和痛苦却没有任何“社会根源”；没有什么他能够否定/反抗的敌人）

## 连贯性、一致性

当生产达到一定发展阶段时，禁令的规训法则，或者说是其否定模式，便达到极限（起到阻塞的效果妨碍继续发展）。 
为了进一步扩大生产，规训范式必须由功绩范式，或曰“能够”的肯定性模式来取代。因此社会集体无意识由“应当”转向“能够”。
--> 将生产最大化的渴望注入社会集体无意识之中。
