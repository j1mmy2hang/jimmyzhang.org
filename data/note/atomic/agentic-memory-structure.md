1. Input enters the <u>sensory layer</u>, gets filtered, and is tagged
2. <u>Short-term memory</u> holds the topics temporarily
3. If relevance persists, <u>long-term memory</u> encodes it permanently
4. <u>Memory managers</u> periodically refine, compress, and prune memory
5. <u>Core memory</u> anchors everything, shaping how the agent perceives, recalls, and adapts
