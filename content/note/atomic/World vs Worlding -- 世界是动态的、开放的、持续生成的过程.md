## 古典观念：世界 = “物”（A Thing）

静态的、预先存在的、三维的容器
客观的“是”（Being）
我们被“抛入”其中，只能去“解释”它
牛顿主义的、经典物理学的、人类主义的观念

- **一个现成的容器**：我们常说“我来到这个世界上”。这感觉就像，“世界”是一个已经布置好的舞台或者一个巨大的房子，而我们只是后来才进入的演员或住户。它在我们来之前就存在，在我们走之后也还会继续存在。

- **一个客观的对象**：这个“世界”有它自己客观的、不以我们意志为转移的规律（比如物理定律）和社会规则。我们的任务是去**认识它、解释它、适应它**。科学家研究它，我们生活在其中。

- **静态的“存在”（Being）**：它关注的是“世界**是**什么”。这是一个关于实体和本质的问题。比如，现代科学试图描绘一个“世界”的蓝图，福山则认为自由民主是“世界”历史的最终形态。这些都是在给一个**已经完成**的“世界”下定义。


## 量子观念：世界是“过程”（A Process）

动态的、开放的、持续生成的“过程”（Becoming）
世界并非一个完成品，而是由无数能动者的“内行动”所共同编织的网络

世界本质上是可变的、不确定的。历史没有终点，现实充满了各种未被实现的“潜在性”。
  
“世界”不是一个现成的舞台，而是所有能动者（人类与非人类）在持续的“内行动”中，共同编织、共同创造出来的动态过程。

1. **世界在“形成中”（Becoming）**：世界永远不是一个完成品，它时时刻刻都在被（重新）创造。它关注的是“世界**如何形成**”。书里说，“世界，实际上是‘世界化成’的一个切片”。我们看到的稳定世界，只是这个动态过程的一个瞬间快照。

2. **我们是参与者和塑造者，而非旁观者**：我们不是被动地生活在一个给定的世界里，而是**主动地参与到“worlding”这个过程中**。我们的每一个行动，每一次互动，每一次思考和言说，都在为这个世界的形成贡献一份力量。

### Worlding = wording/coding + playing/intra-acting

- **wording/coding（语词化/代码化）**：世界的**结构性框架** -- 游戏的规则手册或程序的底层代码，由语言、符号、法律、文化等构成。

- **playing/intra-acting（玩/内行动）**：让世界“活”起来的**动力**。如果没有所有能动者（玩家、NPC、物体、动物...）的“玩”和“内行动”，那个框架就只是一堆死气沉沉的规则。正是“玩”这个实践，驱动着世界不断地变化和生成。

框架本身在“玩”中被改变和重塑
“玩”的过程不仅仅是在一个给定的规则下进行，它同时也在**挑战、解释和重写**这些规则。
wording/coding 更像是一条**流动的河床**，而不是坚固的基岩。它为当下的“玩”提供了方向和限制，但同时，“玩”的流水也在不断地侵蚀和改变着河床的形状。

> “世界化成”（Worlding）是一个循环往复、永不终结的动态过程。在这个过程中，“语词化/代码化”（wording/coding）为“玩/内行动”（playing/intra-acting）提供了临时的结构和意义框架；而“玩/内行动”又在不断地实践、检验、挑战和重塑这个框架，从而生成新的结构和意义。两者互为因果，彼此纠缠，共同驱动着世界的不断“形成”。


## Summary

| 特征        | **World (世界)**    | **Worlding (世界化成)** |
| --------- | ----------------- | ------------------- |
| **词性**    | **名词**            | **动词**              |
| **状态**    | **静态的、完成的、封闭的**   | **动态的、进行中的、开放的**    |
| **本质**    | 一个**容器**，一个**环境** | 一个**过程**，一个**创造活动** |
| **我们的角色** | **旁观者、居住者、解释者**   | **参与者、共同创造者、玩家**    |
| **核心问题**  | “这个世界**是**什么？”    | “这个世界**如何形成**？”     |
| **哲学基础**  | 经典物理学、人类主义        | **量子物理学、后人类主义**     |
|           |                   |                     |
## Significance


1. **赋予我们改变世界的力量**：如果世界是一个固化的“World”，那么改变它几乎是不可能的。但如果世界是一个持续的“Worlding”过程，那么它就**充满了被改变的可能性**。我们的每一个行动，都可能成为改变世界走向的那个“关键变量”。

2. **强调我们的责任**：既然我们都是“世界化成”的参与者，那么我们就对这个世界的样貌负有不可推卸的责任。书里说**“世界兴亡，玩家有责”**，正是这个意思。我们不能再把问题归咎于一个外在的、客观的“世界”或“体制”，因为我们本身就是这个体制的共同塑造者。

**政治的本质，就是参与“worlding”的实践**，就是去“改变关于改变的诸种可能性”。
玩游戏、写评论、进行科学研究、甚至日常的每一次选择，都具有了深刻的政治意义。
